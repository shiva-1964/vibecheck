# ⚡ VibeCheck — AI Powered Toxicity Analyzer & Positive Communication Assistant

### BCA Final Year Project | Flask + SQLite + AI Powered Communication Analysis

---

## 📌 Project Overview

VibeCheck is an AI powered web application designed to detect toxic online communication, analyze emotional tone, and encourage healthier digital interactions through intelligent rewrite suggestions.

The system helps users identify harmful communication patterns in social media comments, chats, and online conversations while promoting positive communication behavior.
